Hi, Thanks for purchasing our template
Boekoe is business landing page, it's focused on book publisher and also can be used for others business types. Boekoe template used UIKit framework for flexibility and agility, and developed with HTML5 and CSS3 for modern web standard. Also, there's 2 homepage layout variations that can be switched with your feels and looks.

The Package

Boekoe download package contains of 3 folder, the folder is "documentation", "html" and "psd". The main file of html file is under folder "html", and here the structure of file from of Boekoe html.

HTML Structure

The layout in Boekoe is flexible because this use "grid" system based on uikit, so you can make any configuration of column (please refer the "http://getuikit.com/docs/grid.html" for some sample of column). Navigation menu is within a nav with an id of "idz-nav" and mobile navigation menu is within a nav with an id of "mobile-nav". The general template structure is the same throughout the template. Visual Structure

The page have four main blocks : header, content-area, before-footer and footer each block consisting of each sections and element present in it.

Logo

To edit the logo you can simply replace the "logo.png" for normal logo position and "logo-sticky.png" for secondary logo when sticky menu active, it's happened when scroll down the page, the logo files located in images folder. The recommended logo height size us 50px.

Menu

Menu at this template is use horizontal menu style, as shown in below image, it's used scrollspy feature from UIKIT to navigate each menu ID's to each sections with the same ID's, please follow below image for detail.

How to make a new Favicon ?

First you must make image on size 32x32 pixel, and save in png format, after that go to this site : http://tools.dynamicdrive.com/favicon/ and create your "favicon.ico" there and replace the old "favicon.ico" in image folder with your new favicon.ico
